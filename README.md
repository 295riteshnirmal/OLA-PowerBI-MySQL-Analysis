OLA Ride Analysis Dashboard using Power BI and MySQL

Project Description

OLA Ride Analysis Dashboard project developed using Power BI and MySQL to analyze ride bookings, revenue, cancellations, customer behavior, and performance insights. The dashboard provides interactive visualizations and data-driven reports for better business understanding and decision-making.

Project Objectives

.Analyze ride booking trends

.Track successful and cancelled rides

.Measure customer and driver ratings

.Analyze payment methods

.Identify high-value customers

.Create interactive business dashboards
